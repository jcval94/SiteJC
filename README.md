# SiteJC

## GitHub Pages

Este repositorio está listo para desplegarse en GitHub Pages porque la página principal vive en `index.html`.

### Pasos rápidos
1. En GitHub, ve a **Settings → Pages**.
2. En **Source**, selecciona **Deploy from a branch**.
3. Elige la rama principal (por ejemplo `main` o `master`) y la carpeta raíz (`/`).
4. Guarda los cambios y espera a que se publique el sitio.
